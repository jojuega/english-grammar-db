### ¿Por qué el sistema americano tiene más puntos?

**No es que use más gramática** — es que el American ESL tiene **mayor granularidad**. Cada punto está desglosado con más detalle (ej: "Present simple: affirmative with he/she/it + -s" se descompone más finamente). Ambos cubren exactamente los mismos dominios gramaticales (30/30 verificados ✅).
